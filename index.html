A Few of my projects

Just a bunch of some of the small projects which i did either because i was bored or as a part of an assignment for my course work.

The Finance App

Built a Finance app using Flask framework as a part of course project with a few other team mates. The webapp uses the concepts of decentralized technology and integrates a simple block-chain technology with out own cryptocurrency along with set of algorithms devised to reduce the number of transactions between the peers.

Learn more

JobsHub

Created a user friendly, scaleable and production ready web application built on the MERN stack for a job search portal meant to be used by applicants and employers to discover jobs and simplify the overall process of recruitment.

Learn more

Hush

Built a fully functional user-defined interactive shell written in C using the Linux system calls. The Shell handles all basic functionalities such as creating processes, basic shell commands, handles background and foreground processes and many more special utility functions along with signal handling.

Learn more

Intra-Day Trador

Built an LSTM Model which learns the trends in stock prices for intra-day trading.

Learn more

BlockBreakerVII

A Game completely built upon OOPs principles and print statements inspired from the popular game Atari BreakOut.

Learn more

SpaceOdyssey

A simple obstacle dodging game in space theme built in python using PyGame library.
